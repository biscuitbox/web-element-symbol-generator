# 원소 타일 생성기 (OpenSCAD + 정적 웹)

## 현재 동작 상태 (2026-05-03 기준)
- 주기율표(전체 118 원소)를 웹에서 클릭하여 선택할 수 있습니다.
- 선택된 원소만 `.scad` 파일로 ZIP 다운로드됩니다.
- 생성된 각 SCAD 파일은 **독립형(standalone)** 모델 코드라서 외부 `use/include` 의존성이 없습니다.
- 원소 이름은 **한글**로 들어가고, 원자량도 함께 기록됩니다.
- 폰트는 한글 깨짐 방지를 위해 `NanumGothic:style=Regular`로 고정되어 있습니다.
- 생성 옵션에서 아래 크기를 조절할 수 있습니다.
  - 타일 크기
  - 원소기호 크기
  - 원소이름 크기
  - 원자번호 크기
  - 원자량 크기
- ZIP 안에는 STL 일괄 변환 스크립트가 함께 포함됩니다.
  - `convert_to_stl.bat` (Windows)
  - `convert_to_stl.sh` (macOS/Linux)

---

## 사용 방법
1. `index.html`을 브라우저에서 엽니다.
2. 주기율표에서 원하는 원소를 클릭해 선택합니다.
3. 크기 옵션(mm)을 조절합니다.
4. `SCAD ZIP 만들기`를 클릭합니다.
5. ZIP을 풀고 STL 변환 스크립트를 실행합니다.

### STL 출력 경로
- 스크립트를 실행한 폴더 아래 `stl` 하위 폴더에 생성됩니다.

---

## Windows 변환 스크립트 주의사항
- `convert_to_stl.bat`는 기본적으로 아래 OpenSCAD 경로를 사용합니다.
  - `C:\Program Files\OpenSCAD\openscad.exe`
- 설치 경로가 다르면 배치 파일 안의 `OPENSCAD_EXE` 값을 수정해야 합니다.

## 비개발자용 빠른 실행 가이드 (Windows)
1. ZIP 파일을 다운로드하고 압축을 풉니다.
2. `index.html` 파일을 더블클릭해서 웹페이지를 엽니다.
3. 원하는 원소를 클릭해서 파란색으로 선택합니다.
4. `SCAD ZIP 만들기` 버튼을 눌러 파일을 받습니다.
5. 새 ZIP을 다시 풀고 `convert_to_stl.bat`를 실행합니다.
6. 같은 폴더의 `stl` 폴더에서 STL 파일을 확인합니다.

### 잘 안 될 때 체크리스트
- OpenSCAD가 설치되어 있는지 확인
- OpenSCAD 설치 경로가 `C:\Program Files\OpenSCAD\openscad.exe`와 다르면 `convert_to_stl.bat` 내부 경로 수정
- 압축을 푼 폴더 안에서 배치파일 실행했는지 확인
- 파일명/경로에 특수문자가 너무 많으면 영문 경로로 옮겨서 다시 실행

---

## 비개발자용 빠른 실행 가이드 (macOS/Linux)
1. ZIP 파일을 다운로드하고 압축을 풉니다.
2. `index.html`을 브라우저로 열어 원소 선택 후 SCAD ZIP을 생성합니다.
3. 생성된 ZIP을 풀고 터미널에서 해당 폴더로 이동합니다.
4. 아래 명령 실행:

```bash
chmod +x convert_to_stl.sh
./convert_to_stl.sh
