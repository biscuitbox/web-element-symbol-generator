# 웹 배포 버전 사용 안내문 (비개발자용)

이 안내문은 **이미 배포된 웹페이지**를 사용하는 분을 위한 설명입니다.
코드를 설치하거나 수정하지 않아도 됩니다.

---

## 1) 준비물
1. 인터넷 브라우저(Chrome/Edge 권장)
2. OpenSCAD 설치 (필수)

---

## 2) 가장 중요한 주의사항 (필수)
Windows에서 STL 변환 배치파일을 사용할 때,
**OpenSCAD 설치 경로는 기본 경로를 사용해야 합니다.**

- 기본 경로:
  `C:\Program Files\OpenSCAD\openscad.exe`

만약 다른 경로에 설치했다면, 생성된 `convert_to_stl.bat` 파일 안의 경로를 직접 수정해야 합니다.

---

## 3) 사용 순서
1. 배포된 웹페이지에 접속합니다.
2. 주기율표에서 원하는 원소를 클릭합니다.
   - 선택되면 배경색이 바뀝니다.
3. 크기(mm) 옵션을 조절합니다.
4. `SCAD ZIP 만들기` 버튼을 누릅니다.
5. 다운로드된 ZIP 파일을 **압축 해제**합니다.
6. STL 변환:
   - Windows: `convert_to_stl.bat` 실행
   - macOS/Linux: `convert_to_stl.sh` 실행
7. 같은 폴더 안의 `stl` 폴더에서 STL 파일을 확인합니다.

---

## 4) Windows에서 STL이 안 만들어질 때
아래 순서대로 확인하세요.

1. OpenSCAD가 설치되어 있는지 확인
2. OpenSCAD 경로가 기본 경로인지 확인
   - `C:\Program Files\OpenSCAD\openscad.exe`
3. ZIP 압축을 푼 폴더 안에서 `convert_to_stl.bat`를 실행했는지 확인
4. 실행 후 뜨는 메시지에서 오류 문구 확인

---

## 5) 자주 묻는 질문
### Q1. 웹에서 바로 STL을 받을 수 없나요?
현재 방식은 먼저 SCAD를 만들고, 로컬 OpenSCAD로 STL을 변환합니다.

### Q2. 생성된 STL은 어디에 있나요?
배치파일/쉘스크립트를 실행한 폴더의 `stl` 하위 폴더에 있습니다.

### Q3. 한글이 깨질 수 있나요?
폰트는 나눔고딕 기준으로 고정되어 있어 대부분 환경에서 안정적으로 표시됩니다.

---

## 6) 한 줄 요약
웹에서 원소 선택 → SCAD ZIP 다운로드 → OpenSCAD 변환 스크립트 실행 → `stl` 폴더 확인
